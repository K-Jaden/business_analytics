# 선형보간 처리 기록

## 선형보간이란?
결측 또는 신뢰도가 낮은 월의 가격을 **앞뒤 정상 데이터의 평균**으로 채우는 방법.
- 단독 1개월: `(전월 + 후월) / 2`
- 연속 N개월: 양끝 정상값 사이를 N+1 등분

## 처리 기준 (CLAUDE.md)
- `tx_count < 3`인 달 → 선형보간
- `tx_count < 3`이 연속 3개월↑ → 해당 기간 제외 검토 (보간 아님)
- `tx_count = 0`인 달 → 선형보간 후 `tx_count=0` 표시 유지, 논문 limitation 명시
