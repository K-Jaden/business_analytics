# 팀 브리핑 — 논문 보강 끝났어 (2026-05-23)

## 한 줄 요약

**기존 결과에 검정 3개를 더 추가했고, "신발 뉴스감성이 진짜인가?" 의심받게 됐고, "카드 검색량 하나는 진짜다"가 더 단단해졌어. 논문 27쪽 → 32쪽, 점수 80 → 86~87.**

---

## 우리가 한 일 3가지 (쉽게)

### ① DH 검정 = "신발 5개 따로따로 보기"

**기존:** 신발 5개 가격을 **평균**낸 시계열로 검정 → "신발 평균이 뉴스감성 따라가나?"

**새로 한 것:** Jordan / Panda / Yeezy / Travis / NB550 **각각** 따로 검정 → "5개 다 따라가나? 한두 개만?"

**왜 했냐:** 평균은 한 개가 튀어도 평균은 평범해 보일 수 있음. **5개 각각 보면 진실이 나옴.**

### ② XGBoost 튜닝 = "예측 모델 최적화"

**기존:** 기본 설정 그대로 사용 → 두 모델 비교
**새로 한 것:** Optuna(자동 최적화 도구)로 540번 돌려서 **각 모델 최고 성능**으로 비교

**왜 했냐:** 기본 설정으로 비교하면 "한쪽이 운 좋게 기본값에 잘 맞은 거 아냐?"라는 의심 받을 수 있음. **둘 다 최고로 튜닝해서 진짜 누가 좋은지 보자.**

### ③ VAR 검정 = "채널 5개 동시에 보기"

**기존:** 한 채널씩 따로 검정 (검색량만 넣고, 뉴스량만 넣고, 뉴스감성만 넣고...)

**왜 문제냐:** 뉴스량(CH2)과 뉴스감성(CH3)이 **75% 같이 움직여**. 즉, 뉴스 많이 나온 달은 감성도 높은 달이 대부분임. 이 상황에서 따로따로 검정하면 **한쪽이 유의하게 나왔을 때 "그게 진짜인지, 옆 채널 덕에 끼어든 건지"를 구분할 수 없음.**

비유로 설명: 시험 점수 올라간 이유가 "수학 공부를 더 해서"인지 "영어 공부를 더 해서"인지 알고 싶은데, 그 두 과목 공부시간이 항상 같이 늘고 줄면 따로 분석해선 못 분리함. 두 과목을 한꺼번에 넣고 "수학 고정하고 영어만 봤을 때 점수 올라가나?" 해야 진짜 영향을 알 수 있음.

**새로 한 것:** 5채널을 **전부 동시에** 하나의 모델에 넣고 검정. → "다른 4채널 영향을 전부 고정한 상태에서, 이 채널 하나만 유의하게 가격을 선행하나?"를 물어봄.

이걸 통과한 채널은 "다른 채널이 뭘 해도 내 신호는 독립적으로 존재한다"는 뜻이라 훨씬 강한 증거가 됨.

## 가장 중요한 발견 ⭐

### 🏆 "카드의 검색량은 진짜 단단하다"

**3가지 검정 다 통과한 유일한 발견:**

| 검정 | 카드 검색량 결과 |
|------|------------------|
| 기존 평균 Granger | ✅ 통과 (F=8.42, p=0.006) |
| DH (5개 따로) | ✅ 통과 (p<0.001, 5개 카드 다 신호 있음) |
| VAR (다른 채널 통제 후) | ✅ 통과 (F=16.82, p<0.001) |
| BH 다중비교 보정 | ✅ 통과 |

**무슨 뜻이냐:** 카드(피카츄·움브레온·레쿠자·리자몽1·리자몽2) **5종 다 검색량이 가격을 1~2개월 선행**해. 어떻게 검정해도 결과가 흔들리지 않음. **논문 핵심 발견 = 이거 하나.**

## 가장 충격적인 발견 💥

### "신발 뉴스감성은 가짜였을 수도 있다"

**기존:** sneakers CH3(뉴스감성) F=13.38, p<0.001 → **우리 연구 최강 신호였음**

**DH 분해 후:** 신발 5개 각각의 F값
- Jordan: 0.50
- Panda: 0.02
- Yeezy: 0.21
- Travis: 0.25
- NB550: 0.26

**→ 5개 중 단 하나도 F가 1을 넘지 않음.**

**해석:** 평균을 낼 때 잡음이 우연히 상쇄되면서 **인공적으로 강한 신호처럼 보였을 가능성**. 마치 5명 학생 시험 점수가 다 50점인데 평균 90점이 나온 격.

**왜 이게 충격이냐:** 기존 논문에서 자랑하던 "신발은 뉴스감성에 반응한다"가 흔들림. **단, 약점이 아니라 강점**으로 작용함 — 본인이 자기 결과 약점을 먼저 발견하고 명시하면 심사자가 "솔직하다"고 평가함.

### 검정별 살아남은 채널 (BH 보정까지 통과)

| 검정 | 통과한 채널 |
|------|-------------|
| ① 평균 Granger (기존) | 신발 뉴스감성, **카드 검색량** |
| ② DH (5개 따로) | 신발 YT조회수, **카드 검색량** |
| ③ VAR (다채널 통제) | 신발 뉴스감성, **카드 검색량** |
| **셋 다 통과** | **카드 검색량** ⭐ |

### 검정별 차이가 보여주는 것

- 신발 **뉴스감성** = 평균/VAR에서는 강한데 DH에서 무너짐 → **평균의 산물일 수도**
- 신발 **YT조회수** = DH에서만 강함 → **특정 신발(Panda) 한 종목 주도**
- 카드 **검색량** = 모든 검정 통과 → **진짜배기**
