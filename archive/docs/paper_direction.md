## 옵션 A — 쉽게 말하면

> **"지금 논문에 회귀 실험을 옆에 추가하는 것"**

지금 논문은 "채널이 가격이 오를지 내릴지(방향)를 예측하는가"를 연구해.  
여기에 "채널이 가격이 얼마인지(수준)도 예측하는가"를 나란히 붙이는 방식.

- 추가 작업: 회귀용 DM 검정 1개만 추가
- 결론 형태: "방향 예측은 채널 선택 무관, 수준 예측은 채널 많을수록 유리"
- 난이도: 쉬움
- 임팩트: 무난함

---

## 옵션 B — 쉽게 말하면

> **"미디어는 가격이 얼마인지는 알지만, 오를지 내릴지는 모른다 — 왜?"를 핵심으로 바꾸는 것**

지금 SHAP 결과를 보면 신기한 패턴이 있어:
- 분류(방향 예측): 채널 기여 거의 없음. 가격 패턴이 전부.
- 회귀(수준 예측): 채널이 1~3위 독점 (특히 레고에서 Google Trends 압도적)

이 발견을 논문의 **핵심 주장**으로 만드는 방식.  
RQ를 4개로 재구성하고 "기존 연구와 다른 각도"로 포지셔닝.

- 추가 작업: MAPE 변환 + 레고 robust 확인 + DM 2개 + Granger-SHAP 비교 테이블
- 결론 형태: "미디어 채널은 방향보다 수준에 기여하며, 자산마다 메커니즘이 다르다"
- 난이도: 중간
- 임팩트: 블루오션 가능

## 어느 쪽이 나을까?

| | 옵션 A | 옵션 B |
|--|--------|--------|
| 2주 안에 가능? | ✅ 여유 있음 | ✅ 빡빡하지만 가능 |
| 추가 작업 | DM 검정 1개 | 아래 체크리스트 |
| 논문 차별성 | 보통 | 높음 |
| 리스크 | 낮음 | 중간 |

**→ DM 검정 결과 보고 결정. DM이 유의하면 B, 유의하지 않으면 A가 안전.**

## 지금 결과에서 나온 핵심 발견 (논문 소재)

### 발견 1: 방향 vs 수준 비대칭
```
분류 SHAP (방향 예측):
  3개 자산 모두 → price_vs_ma3 압도적 1위, 채널 기여 거의 없음

회귀 SHAP (수준 예측):
  sneakers → CH1(Google Trends), CH5(YT감성)이 1,2위
  lego     → CH1이 0.33으로 압도적 1위 (전체 변수 중 최고)
```

### 발견 2: Granger ≠ SHAP (인과 ≠ 예측 기여)
```
sneakers:
  Granger 유의 = CH3(뉴스감성), CH4(YT조회수)
  회귀 SHAP 상위 = CH1(Google Trends), CH5(YT댓글감성)
  → 완전 불일치

lego:
  Granger 유의 = CH1(marginal)
  회귀 SHAP 상위 = CH1 압도적
  → 일치
```

### 발견 3: 자산별 지배 채널 차이
```
스니커즈: 뉴스감성(CH3), YT(CH4) → buzz 주도
카드:     Google Trends(CH1) → 검색 관심도 주도
레고:     Google Trends(CH1) → 장기 트렌드 주도
```
